# Contributing

Thank you for your interest in the Crypto Safety Checker!

Contributions are welcome.

## How to Contribute

You can help improve this project by:

- Reporting incorrect behavior
- Suggesting improvements
- Improving documentation
- Adding new safe test scenarios
- Improving investigation logic
- Fixing typos or formatting

## Before Opening an Issue

Please include:

- A clear description of the problem
- The expected behavior
- The actual behavior
- Steps to reproduce (if applicable)

## Safety Rules

For security reasons, please do **NOT** submit:

- Private keys
- Seed phrases
- Wallet passwords
- Personal information
- Malicious payloads
- Active phishing links

If you want to demonstrate a phishing example, please sanitize or obfuscate it.

## Pull Requests

Please keep pull requests:

- Focused
- Well explained
- Easy to review

Small improvements are preferred over very large changes.

## Project Goals

This project aims to:

- Help users investigate Web3 resources
- Separate verified facts from unknowns
- Encourage safe decision-making
- Never guarantee safety

Thank you for helping make Web3 safer.
